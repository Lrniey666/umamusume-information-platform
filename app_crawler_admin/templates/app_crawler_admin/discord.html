Discord Bot 指令

在任意 Discord 頻道輸入以下指令（Bot 需已上線）：

!channel help 顯示所有指令說明
!channel set crawl [名稱] 設當前頻道為爬取頻道
!channel set news [名稱] 設當前頻道為推播頻道
!channel list 列出所有頻道設定
!channel remove 移除當前頻道設定
Bot 指令與此頁面的設定共用同一個資料庫，兩端皆可管理，即時生效。
