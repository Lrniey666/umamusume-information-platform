💬 Discord Bot 指令

在任意 Discord 頻道輸入以下指令：

!channel help 顯示所有指令說明
!channel set crawl 設為爬取頻道
!channel set news 設為推播頻道
!channel list 列出所有設定
!channel remove 移除當前頻道
名稱可選填：
!channel set news 每日新聞版
🚀 啟動方式
本地啟動
python manage.py run_discord_bot
Docker 啟動
docker compose up discord-bot
必要環境變數： DISCORD_BOT_TOKEN — 頻道設定請透過此頁面或 Discord 指令管理，無需在 .env 設定頻道 ID。
