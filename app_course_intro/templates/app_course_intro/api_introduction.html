🔌 API 技術介紹

本平台整合以下四項外部 API，構成完整的 AI 分析與互動功能。

✨
Google Gemini API
ai.google.dev — 生成式 AI 核心引擎
gemini-3.5-flash gemini-embedding-001
主要用途
• 公告情緒分類標記（正/中/負）
• 關鍵詞 LLM 分析報告生成
• RAG 知識庫問答（Embedding + 生成）
• Agentic RAG 多輪對話推理
• Function Calling 工具決策
使用限制
  • 免費層：15 RPM / 1M TPM
  • Embedding 每次最多 2048 tokens
  • Flash 模型上下文 1M tokens
  • 須申請 API Key（ai.google.dev）
使用方式
環境變數：GEMINI_API_KEY

Python SDK：
from google import genai
client = genai.Client(api_key=...)

呼叫：client.models.generate_content()
🧠
Anthropic Claude API
api.anthropic.com — 高品質長文推理模型
claude-sonnet-4-6 claude-3-5-sonnet
主要用途
• 深度情緒分析與報告生成
• 複雜推理類 LLM 報告
• 多輪對話 AI Agent（備援）
• 長文本摘要與整理
使用限制
  • 需付費 API 金鑰
  • Sonnet: 200K context window
  • Rate limit 依方案而定
  • 台灣需透過代理或 VPN
使用方式
環境變數：ANTHROPIC_API_KEY

Python SDK：
import anthropic
client = anthropic.Anthropic()

呼叫：client.messages.create()
▶
YouTube Data API v3
developers.google.com/youtube — 影片數據查詢
search.list videos.list commentThreads.list
主要用途
• 搜尋賽馬娘相關影片
• 取得影片統計數據（觀看/讚）
• 抓取影片留言進行情緒分析
• Discord Bot 嵌入影片資訊
使用限制
  • 每日配額：10,000 units
  • search.list 消耗 100 units/次
  • videos.list 消耗 1 unit/次
  • 需 Google Cloud Project
使用方式
環境變數：YOUTUBE_API_KEY

Python：
from googleapiclient.discovery import build
yt = build('youtube', 'v3', developerKey=...)

搜尋：yt.search().list(q=...).execute()
💬
Discord Bot API
discord.py — 即時互動機器人框架
discord.py Slash Commands Embeds
主要用途
• 在 Discord 頻道推播最新公告
• Slash 指令觸發 RAG 問答
• 嵌入卡片展示角色資訊
• 情緒分析結果通知
使用限制
  • 需 Discord Developer Portal 申請
  • Bot 需被邀請加入伺服器
  • Rate limit: 50 requests/sec
  • Gateway Intents 需手動開啟
使用方式
環境變數：DISCORD_BOT_TOKEN

Python SDK：
import discord
bot = commands.Bot(command_prefix='!')

事件：@bot.event / @bot.slash_command()
